# Live Rulebook

Each Rule is one real-CLI guarantee — a flow or agent driven against the real `claude`/`codex` CLI and a real
subscription, proving what only a live agent can. Live tests are opt-in: they skip unless `RUN_LIVE=1`
(`[LiveFact]`), so `dotnet test` and CI never hit a real CLI. Convention, parity discipline, the Rule shape, and
the going-forward adoption policy live in [`../../../Harness/README.md`](../../../Harness/README.md) and
`template.md`.
